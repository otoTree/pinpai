# 逆袭爽剧 · 内流控制默认模板（10 集）

## 模板定位
- 题材：草根逆袭 / 重生复仇 / 强爽点
- 核心目标：最大化观看时长 + 强追更率
- 情绪机制：长期压抑 → 延迟满足 → 阶段性翻身 → 再压抑 → 终极逆转

---

## 1. Payoff Ledger（爽点账本）

### 爽点预算
```json
{
  "S": 2,
  "A": 3,
  "B": 5
}
```

### 爽点语义约束
- S：身份揭晓 / 阶级逆转 / 终极反杀
- A：公开打脸 / 资源夺回
- B：暗爽 / 私下反击

---

## 2. Suppression Index（压抑指数）

### 权重配置
```json
{
  "protagonist_humiliation": 0.25,
  "villain_advantage": 0.25,
  "truth_hidden": 0.2,
  "reward_delayed": 0.2,
  "status_gap": 0.1
}
```

### 阈值
- A / S 释放前：最近 2 集平均压抑 ≥ 0.65
- 压抑 < 0.45 且无爽点 → 判定失败

---

## 3. Hook Contract（钩子）

### 允许类型
- reward_delay（最高优先级）
- information_gap
- power_shift

### 示例
```json
{
  "hook_type": "reward_delay",
  "withheld_payoff": "真实身份曝光",
  "payoff_level": "S",
  "scheduled_release": 6
}
```
