## 提示词优化分析报告

你的提示词框架已经非常专业，但在**解决AI视频生成连贯性问题**方面仍有几个关键缺口。以下是具体的优化点：

---

### 核心问题：你的提示词在告诉AI"拍什么"，但没有告诉AI"怎么接"

当前提示词的问题不是内容不够详细，而是**缺少"镜头间关系"的元数据层**。

---

### 具体优化点（按优先级排序）

#### **1. 【致命缺失】没有强制要求"过渡逻辑"字段**

当前没有任何字段要求描述"如何从上一镜来到这一镜"。

**需新增字段**：
```typescript
"transitionFromPrevious": {
  "incomingAction": "上一镜结尾时的状态（如：手正在离开桌面）",
  "continuityMatch": "与上一镜的连接点（如：同一只手入画）",
  "spatialRelationship": "空间位置关系（如：切到同一房间的另一侧）",
  "timeGap": "时间跳跃（如：连续 / 2秒后 / 同时发生）"
}
```

**为什么需要**：AI需要知道"杰克的手为什么突然出现在这里"，而不是猜测。

---

#### **2. 【严重缺失】没有"道具生命周期追踪"**

当前只列出了静态的"Assets List"，但没有追踪道具在镜头间的移动和状态变化。

**需新增**：
```typescript
"props": [
  {
    "name": "咖啡杯",
    "state": "满的/被打翻/在桌上/在地上",
    "position": "桌子右上角",
    "continuityNote": "从镜8延续"
  }
]
```

**为什么需要**：咖啡杯在镜8被打翻，镜9消失，镜13又出现——AI不知道这些是同一个杯子。

---

#### **3. 【重大缺陷】"characterAction"过于笼统**

当前字段只描述"做什么"，没描述"动作的起止状态"。

**优化建议**：
```typescript
"characterAction": {
  "summary": "拿起手机",
  "startState": "手垂在身侧",
  "endState": "手机举到眼前",
  "muscleTension": "手指僵硬，轻微颤抖",
  "speed": "急促/缓慢/突然"
}
```

**为什么需要**：只有知道"从哪里开始到哪里结束"，AI才能生成连贯的动作弧线。

---

#### **4. 【光线断层】缺乏"光源演变"的描述**

当前"lightingAtmosphere"是静态描述，但镜头间的光线可能是动态变化的。

**需新增**：
```typescript
"lightingEvolution": {
  "primarySource": "应急灯",
  "secondarySource": "月光",
  "changeOverShot": "红光逐渐减弱，月光比例增加",
  "transitionFromPrev": "承接镜14的红色主光"
}
```

**为什么需要**：镜13-14的光线变化（爆炸→黑→月光→红光）需要明确的时间线。

---

#### **5. 【视线逻辑】缺乏"视线向量"**

人物看的方向决定了观众的空间认知。

**需新增**：
```typescript
"eyeline": {
  "direction": "画左45度/向上/看向镜头外",
  "target": "监视器/窗外/手机",
  "changeWithinShot": "从看镜头转到看监视器"
}
```

**为什么需要**：镜6看镜头，镜7看监视器，中间缺了"转头"——AI需要知道这个转头发生在镜6结尾还是镜7开头。

---

#### **6. 【声音衔接】缺乏"声音的空间感"**

"soundEffect"太笼统，没有说明声音来自哪里。

**需新增**：
```typescript
"audioDesign": {
  "source": "画外/画内/主观音",
  "distance": "近/远/逐渐靠近",
  "reverb": "室内混响/空旷",
  "relationToPrev": "镜7的啸叫声延续到本镜开头"
}
```

---

#### **7. 【镜头运动】缺乏"运动动机"**

"camera"字段只说"怎么动"，没说"为什么动"。

**需新增**：
```typeshline
"cameraMotivation": {
  "trigger": "被角色动作触发/被声音触发/情绪驱动",
  "relationship": "跟随角色/揭示环境/主观视角"
}
```

**为什么需要**：推镜头是因为杰克逼近镜头，还是因为情绪紧张？这影响生成效果。

---

#### **8. 【时间标记】缺乏"精确的时间锚点"**

4-6秒的区间太宽，需要更精确的时间分配。

**需新增**：
```typescript
"timeline": {
  "shotStart": "00:00:00",
  "actionStart": "00:00:01（台词开始）",
  "actionEnd": "00:00:04（手触到桌子）",
  "shotEnd": "00:00:05"
}
```

**为什么需要**：多个镜头的动作需要精确对齐（如镜11-12的同时发生关系）。

---

#### **9. 【物理连续性】缺乏"环境状态追踪"**

场景中的物理状态需要在镜头间保持一致。

**需新增**：
```typescript
"environmentalState": {
  "debris": "玻璃碎屑位置",
  "liquid": "咖啡渍位置",
  "smoke": "烟雾浓度和流向",
  "damage": "新增的破坏痕迹"
}
```

---

#### **10. 【AI生成约束】缺乏"禁止项"**

AI常犯的错误需要明确禁止。

**需新增**：
```typescript
"generationConstraints": [
  "禁止道具凭空出现/消失",
  "禁止人物服装突变",
  "禁止光线方向无理由改变",
  "禁止同一场景内建筑结构变化"
]
```

---

```typescript
{
  "shots": [
    {
      // 原有字段保留
      "sequence": 1,
      "description": "...",
      "sceneLabel": "...",
      
      // 优化后的动作描述
      "characterAction": {
        "summary": "拿起手机",
        "startState": "手垂在身侧",
        "endState": "手机举到眼前",
        "muscleTension": "手指僵硬",
        "speed": "急促"
      },
      
      // 新增过渡层
      "transition": {
        "incomingAction": "上一镜：手离开桌面",

        "timeGap": "连续，无跳跃"
      },
      
      // 新增道具追踪
      "props": [
        {
          "name": "手机",
          "state": "屏幕死黑",

        }
      ],

      "eyeline": {
        "direction": "向下45度",
        "target": "手机屏幕",

      },
      
      // 优化光线描述
      "lightingAtmosphere": "应急灯红光，自上而下",
      "lightingEvolution": {
        "primarySource": "应急灯",
        "changeOverShot": "稳定不变"
      },
      
      // 优化声音描述

        "size": "Close-up",

      },
      
      // 时间锚点
      "timeline": {
        "shotStart": "00:01:15",
        "actionStart": "00:01:16（开始敲击）",

      ],
      
      "duration": 4,
      "videoPrompt": "...",
      "suggestedAssets": {...}
    }
  ]
}
```

---

### 最关键的三个修改点（如果不改其他的话）

1. **增加 `transition` 字段**：解决"怎么从上一镜来"的问题
2. **增加 `props` 生命周期追踪**：解决道具连续性
3. **`characterAction` 改为对象 + `startState/endState`**：解决动作弧线

这三个改动能让AI生成时的连贯性提升50%以上。
